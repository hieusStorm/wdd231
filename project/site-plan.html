Site Name

Site Name: Magic Deck Learner

This name represents the purpose of the site and provides and shows how the site is for learning how to play different Magic the Gathering decks.

Site Purpose

This site will provide a place to view deck lists for Magic the Gathering along with a brief description of each deck to help with being able to learn how to play each deck.

Scenarios

  1. What makes this system deck building site stand out?
    • This site unlike other deck list sites will provide a description for how to play the deck list provided
  2. Will any of the description include the general archtype(ex aggro, control, combo...) of each deck
    • Yes part of each description will provide the general archtype of the deck including the following deck archtypes aggro, control, combo, midrange, and tempo.
